# Reach-the-Goal-423

We're planning to make a game where a player has to make him save from thorns coming towards him and to reach the top of the arena. 
Arena - 15X15 ground
Movement - Player can move left,right,top,bottom
Thorns - Thorns can appear from 6 grids by loop after 1 second. Horizontally & Vertically.
Camera Movement - camera can be moved up and down.
Shield - There will be 4 guard walls in the ground. Players  can make him save from thorns. But can stay behind it for seconds.
Shield wall movement.
Gameover - If the thorn touches the player, it will be game over. Staying behind walls more than 3 seconds also will cause game over.
Win - If the player can reach the top of the ground, it will be counted as a win.
Score - Covering one grid will count +1 in score. Win will count +10 with the score. 
Speed - Speed of thorns will increase with time.
Small glowing gems appear randomly in the arena.
Collecting one increases the score by +5.
But reaching for them is risky because thorns may block the way.
Instead of instant game over on the first thorn touch, the player has 3 lives.
Each thorn hit reduces life by 1.
When life = 0 → Game Over.
This balances difficulty as thorns get faster.
